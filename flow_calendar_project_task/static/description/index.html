Flow Calendar Project Task

Plan "Project Tasks" in the Odoo Calendar

Benefits of planning in the Calendar:
  • Never miss a Task (deadline) again. It's planned and can be filtered in the Calendar!
  • Filter Calendar Events by (your) Task Stages. E.g. show only Events in stage Analysis and/or Development and/or Testing.
  • Plan a Task right away from the Calendar.
  • See availability of users due to meetings, holidays and all planned project tasks.
  • A Task can be planned as many (linked) Calendar Events.

Extra features:
  • Calendar events can't exceed the Deadline of the Task. Checks on changing deadline and Calendar events.
  • Checks changing the Task owner equals the owner of its Events.
  • Prefixes the Calendar Event name with Sale Order Name (e.g. SO099), in case Task is linked to a Sale Order Line. Otherwise the prefix is "P/<project_id>"

Installation notes

A bit of system adminsitration is required

Python dependency: html2text

Required to convert the Task "HTML description" into plain-text for the Calendar Event description.

Install either by:

  • pip: 
    pip3 install html2text
    - OR -
  • Ubuntu/Debian package: 
    apt-get install python3-html2text
