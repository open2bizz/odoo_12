Mail Messages Draft

This is an optional extension for Mail Messages Easy

Main features

  • Save message as draft to send later.
    Hint: You can have several drafts per one record!
    Each time open you open new mail compose wizard from the same record (e.g. CRM Lead) and press "Save & Close" new draft is created.
    But if you press "Save & Close" while sending you message from drafts, draft you are sending from will be overwritten.
  • Edit draft messages
  • Send message from draft
  • Access control: Users can see and edit only drafts they have created
  • Compose new message directly from messages menu Click to get Pro Version!
  • Restore message from draft in composer. Select which draft version to restore Click to get Pro Version!
  • Add labels to drafts to mark different versions easier Click to get Pro Version!
  • Move draft to other record Click to get Pro Version!
  • Add followers to draft to collaborate on draft Click to get Pro Version!

Support

  • Tested on Odoo Community v14.0
  • Russian language translation
  • English, German and Russian language support
  • Click to contact us for support or any custom app you need

Known issues

  • After message is sent from the draft in Odoo 10 and Odoo 12 you will be automatically redirected to "Messages" view.
    However this option in not working in Odoo 12 yet
